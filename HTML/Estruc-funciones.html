Componentes físicos

  1. Algún dispositivo cómo una computadora, un telefono movil, una tableta entre más... Este elemento es indispensable pues de este dependerá la velocidad internet.
  2. La comunicación entre nuestro ordenador e Internet necesita transportarse a través de algún medio físico o inalámbrico, normalmente se utilizan líneas telefonicas sin embargo ahora es más ocupada la fibra óptica.
  3. El modem el cual es un elemento que permite establecer la conexión física entre nuestro PC y la línea teléfonica o de transmisión.
  4. Servidores que son computadoras de altas prestancias que ofrecen servicios a los usuarios.

Componentes lógicos

  1. El terminal es el elemento que sirve al usuario para recibir y enviar información.
  2. Un dominio de Internet es una red de identificación asociada a un grupo de dispositivos o equipos conectados a la red Internet.
  3. Protocolos, son un conjunto de reglas, el protocolo TCP-Transmisión Control Protocol (Protocolo de control de transmisión)/IP-Internet Protocol (Protocolo de Internet) que permite la comunicación entre equipos.
  4. Dirección Ip; es un número que permite identificar a un dispositivo conectado a la red. Esta compuesto por cuatro conjuntos de números que van del 0 al 255, por ejemplo 192.168.0.1
  5. URL son traducciones de Ip a una dirección,

Capas de internet

  1. Capa de aplicación: en ella corren las aplicaciones de red (la Web, ftp, e-mail, etc.).
  2. Capa de transporte: transfiere datos entre las aplicaciones que corren en 2 o más hosts.
  3. Capa de red: es responsable del routing, es decir, de conducir los paquetes de datos entre los hosts.
  4. Capa de enlace: resuelve el problema de transmitir datos entre cada par de nodos (host-host, host-switch, host-router, router-router, etc.).
  5. Capa física: entiende cómo deben transmitirse los flujos de bits sobre los distintos medios físicos.
