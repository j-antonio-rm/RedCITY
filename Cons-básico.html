¿Qué es internet?

Es un conjunto complejo de redes de computadoras que se encuentran interconectadas a nivel global en el que se comparten información a través del uso estándares.

Objetivos de internet

Cómo objetivo principal es el intercambio de información favoreciendo el aprendizaje y la comunicación entre las personas, actualmente su objetivo es comercial.

¿Cómo se originó?

ARPANET (Advanced Research Proyects Agency Networks) surge 1964, desarrollado por el departamento de defensa de Estados Unidos, que buscaba como fin crear una red de computadoras que uniera a los centros de investigación de defensa en caso de ataques para mantener contacto remotamente y no se interrumpiese su funcionamiento.

En 1990 desaparece ARPANET, dejando sus bases a Internet quien, adoptando ciertos estándares, logra la primera comunicación de consulta de archivos de hipertexto en el servicio WWW (Word Wide Web). Con ello surge y despega lo que hoy conocemos cómo internet.

Componentes de internet

Para que la comunicación entre las computadoras de una red se lleve a cabo será necesaria la presencia de una serie de componentes. En el caso de Internet estos elementos son:

  • Servidores
  • Clientes
  • Software
  • Dirrección IP
  • Dominio
  • Multimedias

Conexiones de internet

  • Líneas telefónicas: Se puede utilizar una línea telefónica que se subdivide en líneas telefónicas convencionales o digitales.
  • Cable (fibra óptica): Asimismo, puede ser una conexión por cable (que implementa señales luminosas en vez de eléctricas que codifican una mayor cantidad de información y a su vez, dicho cable es de un material llamado fibra óptica que le permite transmitir datos entre nodos).
  • Satelital: También, existe la conexión vía satélite (normalmente estamos hablando de un sistema de conexión híbrido de satélite y teléfono para disminuir la congestión presente en las redes terrestres).
  • Redes inalámbricas: A su vez, podemos implementar redes inalámbricas (que sustituyen los cables por señales luminosas infrarrojas u ondas de radio para transmitir información). En suma, dentro de las redes inalámbricas, otro método utilizado es el de LMDS (que es una sigla que significa Local Multipoint Distribution System y consta de utilizar ondas de radio de alta frecuencia).

Diferencia entre web-internet

Internet es una infraestructura con una inmensa red de computadoras alrededor de todo el mundo conectadas entre sí, las cuales se puede comunicarse siempre y cuando esten conectadas ah internet mientras que la web (la World Wide Web) es una enorme colección de páginas que se asienta sobreesa red de computadoras.

Internet nació y existia mucho antes que la web; nadie antes había desarrollado una forma conectar la internet con todos los documentos y datos que ahora hay en ella, hasta que se crea la web en mano de Berners-Le; quien encontró un sistema eficiente de usar esas conexiones creando páginas web para compartir información.

Word Wide Web(www)

La Web es el diminutivo de world wide web o www cuyas tecnologías para su funcionamiento (HTML, URL, HTTP) fueron desarrolladas en el año 1990 por Tim Berners Lee, la cual cumple objetivo la transmisión de datos y información atraves de documentos de hipertexto.

La web usa para la creación de sus páginas web el lenguaje HTML (hypertext markup language) que unificó la forma de buscar y crear información a través de internet. Para poder acceder a esta información es necesario tener acceso a internet y un navegador web.

Hipertexto: es un conjunto estructurado de textos, gráficos, imágenes o sonidos unidos entre sí por enlaces o vínculos (links) y conexiones lógicas.

Página web: son documentos compuestos por textos, imágenes, audios o videos a los que se puede acceder a través de la word wilde web o www empleando un navegador.

Sitio web: conjunto de páginas web que se encuentran relacionadas por el contenido y por un dominio en internet, y que constituyen una gran red de información.

Si no existiera internet, nadie podría comunicarse a través de la World Wide Web porque no habría manera de enviar esos datos. Y sin la World Wide Web, la mayoría de nosotros encontraríamos extremadamente difícil (y mucho más costoso) acceder a toda la información que tenemos disponible hoy día.
